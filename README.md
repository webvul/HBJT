# HBJT

湖北省交通运输厅网上办事平台客户端。

本客户端集成了交通新闻浏览，行政审批指南，用户办件跟踪，申请结果公示，高速规费查询，即时路况信息等实用功能。

嗯，其实功能也不多。

在项目中大量练习使用 ReactiveCocoa，Masonry 等第三方库，实践了一些个人的想法，也留下了很多经验教训。

我的第一个上架项目。
